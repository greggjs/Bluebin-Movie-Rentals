Installation

This section documents installation instructions for BLUEBIN.

  1. Unzip the .zip file Bluebin-Movie-Rentals-master in your computer. It can be downloaded here.
  2. If you're under Windows:
    1. In order to establish a secure connection to the Database (SSH), download and run PuTTY and follow these instructions:
    2. On the Host Name section, type easlnx01.csi.muohio.edu.
    3. Open SSH > Tunnels. Type 3307 and localhost:3307 under port forwarding.
    4. Press Add and press Open. Login in to EASLNX01 with your personal Miami login and password. Keep the session open while you access the database.
  3. If you're under MAC:
    1. In terminal, type: ssh <uniqueid>@easlnx01.eas.muohio.edu -L 3307:localhost:3307
  4. Open the folder Bluebin-Movie-Rentals-master and run BluebinMovieRentals.exe (Windows) or Bluebin Movie Rentals.app (MAC)

BLUEBIN - FIRST USE

 

  • Main Screen

1. This is the main screen that appears when the user opens the program for the first time. The user will need to create a new account in order to be able to use the software.

2. Login is based on phone number and a personal password, both defined in the account creation menu.

3. Administrators can be able to do additional features with the program such as add movies to the database or change quantity, etc.

 

  • Login Screen

1. Here the user is able to Rent a Movie, Return a Movie, Update Profile and Sign-Out.

 

  • Search features

1. The user is able to search for a movie using the Title, Keyword, Genre, Rating, Release Year or Actor.

 

Known Issues

 

There are no transactions in our database to prevent the collisions of queries, but our system is based on the person physically being there at a single unit to query for data, so in reality there would be not be multiple users on the system at once, unless the database being queried covered all units, which it does not.
